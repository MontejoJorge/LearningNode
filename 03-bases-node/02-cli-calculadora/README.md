# Calculadora cli
```
Opciones:
    --help           Muestra ayuda                                  [booleano]
    --version        Muestra número de versión                      [booleano]
    --n1, --numero1  Primer numero para la operacion a realizar
                                                            [número] [requerido]
    --n2, --numero2  Segundo numero para la operacion a realizar
                                                            [número] [requerido]
    -o, --operacion      Operacion a realizar (suma, resta, multiplicaion,        
                       division)
              [cadena de caracteres] [requerido] [selección: "s", "r", "m", "d"]
```
